
## 🤔 고민과 호기심이 많은 개발자
다양한 사람들과 **다양한 고민을 공유하면서 기획과 아이디어가 구체화되는 것**에서 큰 재미를 느끼며 성장중입니다.

보다 나은 서비스 제공에 관심이 많으며, **새로운 기술을 배우는 것을 두려워하지 않습니다.**

호기심에서 출발한 **단기간의 성장 경험**을 발판삼아 앞으로도 꾸준히 성장하고 싶습니다.
